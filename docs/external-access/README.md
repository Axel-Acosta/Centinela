# External access request packet

This folder contains private/share-intentionally materials for requesting authenticated access to external matching systems.

Current target:

- OpenSanctions production API
- Hosted Yente endpoint for entity matching/search

Files:

- `opensanctions-access-request-email.md`
  - polished email text ready to send
- `centinela-private-technical-summary.md`
  - short private architecture and workflow summary to attach or paste if OpenSanctions asks for more context
- `opensanctions-follow-up-checklist.md`
  - questions to answer, credentials to store, and integration steps after access is granted

Do not add secrets, API keys, passwords, or live credentials to these files.
