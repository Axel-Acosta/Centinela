# DNCP to Centinela rule crosswalk

## Purpose

This document records how the current Centinela Paraguay procurement rules relate to Paraguay's DNCP red-flag reality.

It is not a claim that the current Centinela rules are official DNCP rules.

It is a working crosswalk for local compatibility, methodology disclosure, and later public explanation.

## Current interpretation rule

- If a rule says "strong conceptual analogue", it fits local procurement-control logic well but is not yet documented as a direct DNCP public flag equivalent.
- If a rule says "partial local analogue", it is locally useful and directionally compatible but still more Centinela-specific in threshold or structure.
- If a rule says "transparency and interpretability support rule", it helps analysts trust the data layer but should not be presented as a substantive local corruption-risk flag.
