# Generated data lives outside this repo

Centinela writes generated raw data, normalized bundles, reports, and downloaded source files to the local runtime folder by default:

```text
C:\Users\<user>\AppData\Local\Centinela\data
```

This repo keeps only this note so Git/GitHub and sync tools do not track large generated artifacts.

See `docs/ops/workspace-storage.md` for the storage policy.
