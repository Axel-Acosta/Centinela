# Comparative intelligence index

Each reference file captures four things:

1. what the system is genuinely strong at
2. what does not port cleanly to Paraguay
3. what Centinela should adopt now
4. what should remain a later benchmark or optional integration

The project's active execution rule is that no major reference should remain decorative. Comparative work is only complete once it produces a concrete component, artifact, or staged requirement in `docs/execution/reference-to-component-execution-plan.md`.
